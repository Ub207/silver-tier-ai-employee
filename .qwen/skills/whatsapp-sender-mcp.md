# Skill: whatsapp-sender-mcp
Generates a Playwright code snippet to open WhatsApp Web, find the contact, and pre-fill the reply.
Human must press Enter/Send — this skill NEVER auto-sends.

## Trigger
Called by `approval-checker` after a reply is approved, or directly:
`/whatsapp-sender-mcp "[reply text]" "[contact name]"`

---

## Step 1 — Validate inputs
- `reply_text`: the approved reply string (must not be empty)
- `contact_name`: WhatsApp contact name to search for
- Confirm reply is under 1000 characters (WhatsApp soft limit for readability)

If reply text contains `[TIME]`, `[TODAY]`, or `[AMOUNT]` placeholders → STOP and ask human to fill them in before proceeding.

---

## Step 2 — Print the approved reply for final review
```
=== FINAL REVIEW BEFORE SENDING ===
Contact: [contact_name]
Reply:
"[reply_text]"

Character count: [N]/1000
Financial content: [Yes/No]
====================================
Confirm you want to open WhatsApp Web with this reply pre-filled? (yes/no)
```

Wait for human confirmation.

---

## Step 3 — Generate the Playwright MCP code snippet

Output the following Python code block for the human to run or for the MCP server to execute:

```python
"""
WhatsApp Web sender — pre-fills reply for human to review and send.
Generated by /whatsapp-sender-mcp skill.
Contact: {contact_name}
"""
import time
from pathlib import Path
from playwright.sync_api import sync_playwright

REPLY_TEXT   = """{reply_text}"""
CONTACT_NAME = """{contact_name}"""
SESSION_DIR  = "silver_tier/silver_tier/whatsapp_session"

def send_whatsapp_reply():
    with sync_playwright() as p:
        ctx = p.chromium.launch_persistent_context(
            user_data_dir=SESSION_DIR,
            headless=False,
            slow_mo=400,
            args=["--start-maximized"],
        )
        page = ctx.pages[0] if ctx.pages else ctx.new_page()

        print(f"Opening WhatsApp Web...")
        page.goto("https://web.whatsapp.com", wait_until="domcontentloaded")
        page.wait_for_timeout(4000)

        # Search for the contact
        print(f"Searching for contact: {CONTACT_NAME}")
        search_selectors = [
            '[data-testid="chat-list-search"]',
            '[placeholder="Search or start new chat"]',
            'div[contenteditable="true"][data-tab="3"]',
        ]
        for sel in search_selectors:
            try:
                page.wait_for_selector(sel, timeout=5000)
                page.click(sel)
                page.keyboard.type(CONTACT_NAME, delay=100)
                break
            except Exception:
                continue

        page.wait_for_timeout(2000)

        # Click the first matching contact in results
        contact_selectors = [
            f'[title="{CONTACT_NAME}"]',
            f'span[title*="{CONTACT_NAME.split()[0]}"]',
        ]
        for sel in contact_selectors:
            try:
                page.wait_for_selector(sel, timeout=4000)
                page.click(sel)
                break
            except Exception:
                continue

        page.wait_for_timeout(1500)

        # Type the reply into the message box
        print("Typing reply...")
        msg_selectors = [
            '[data-testid="conversation-compose-box-input"]',
            'div[contenteditable="true"][data-tab="10"]',
            'footer div[contenteditable="true"]',
        ]
        for sel in msg_selectors:
            try:
                page.wait_for_selector(sel, timeout=4000)
                page.click(sel)
                page.keyboard.type(REPLY_TEXT, delay=30)
                break
            except Exception:
                continue

        print()
        print("=" * 50)
        print("REPLY PRE-FILLED. REVIEW AND PRESS ENTER TO SEND.")
        print("Close the browser without pressing Enter to CANCEL.")
        print("=" * 50)

        # Keep browser open for 5 minutes for human review
        page.wait_for_timeout(300_000)
        ctx.close()

if __name__ == "__main__":
    send_whatsapp_reply()
```

## Notes
- Session dir uses the existing `whatsapp_session/` — must be authenticated (run `python whatsapp_watcher.py --setup` if needed)
- Always offer manual copy-paste fallback (Step 4, Option 2)
- The snippet expires after use — `whatsapp_sender_temp.py` is regenerated each time
- If `[TIME]` placeholder found in reply, STOP and ask human to replace it first
