ihower left a comment (openai/openai-agents-python#1897)



 my best guess is that the error happened because @tnduyh5 incorrectly passed the handoff into the tools parameter.



For example:

```

 triage_agent = Agent(name="Triage agent", tools=[handoff(refund_agent)]) # ❌

```

 

This will cause:  AttributeError: 'Handoff' object has no attribute 'name' which reproduces exactly the same error described in the original issue.



The correct way is to pass it in the handoffs parameter instead of tools:

```

triage_agent = Agent(name="Triage agent", handoffs=[handoff(refund_agent)]) # ✅

```



@tnduyh5 Hope this helps you!
