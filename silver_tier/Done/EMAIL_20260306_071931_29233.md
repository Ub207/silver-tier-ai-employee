HI, I don't know if you already know about it, but you can get information about the run until the exception was raised accessing `run_data` from every `AgentsException` object (e.g. `MaxTurnsExceeded`):

```python

class AgentsException(Exception):

    """Base class for all exceptions in the Agents SDK."""



    run_data: RunErrorDetails | None



    def __init__(self, *args: object) -> None:

        super().__init__(*args)

        self.run_data = None

```



Otherwise, if you want to avoid the exception for another reason, I have no suggestions.
