Logfire's Python ecosystem integrations - FastAPI, Django, Flask and more



Logfire



Hello!



Logfire is designed to integrate smoothly with the Python ecosystem. That means we have custom integrations with many of the key tools, such as:



* FastAPI

* Flask

* Django

* Requests

* httpx

* SQLAlchemy



And many more that you can find listed on our integrations docs ( https://logfire.pydantic.dev/docs/integrations/#opentelemetry-integrations ).



For example, here's how to instrument a FastAPI app with Logfire:



Install the fastapi extra dependency: pip install 'logfire[fastapi]'



import logfire

from fastapi import FastAPI



app = FastAPI()

logfire.configure()

logfire.instrument_fastapi(app)



@app.get("/hello")

async def hello(name: str):

return {"message": f"hello {name}"}



if __name__ == "__main__":

import uvicorn

uvicorn.run(app)



Once you've done that, you'll start to see data such as FastAPI arguments in your Logfire UI:



FastAPI Example in Logfire UI ( https://logfire.pydantic.dev/docs/integrations/fastapi/ )



Since Pydantic Logfire is OpenTelemetry compatible, it can be used with any OpenTelemetry instrumentation package. You can find the list of all OpenTelemetry instrumentation packages here ( https://opentelemetry-python-contrib.readthedocs.io/en/latest/ ).
