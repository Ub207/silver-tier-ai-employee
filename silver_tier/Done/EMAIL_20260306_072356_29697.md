yashika-git created an issue (openai/openai-agents-python#1680)



When using an agent as a tool, is there a way to access the tool calls made by that agent?



### Example:



```python

from agents import Agent, Runner, WebSearchTool



# Agent with WebSearchTool

internet_search_agent = Agent(

    name="Internet Search Agent",

    model="gpt-4.1",

    instructions="You are a helpful agent that can search internet.",

    tools=[

        WebSearchTool()

    ],

)



# Using WebSearchTool agent as a tool

agent = Agent(

    name="WebOAI website searcher",

    model="gpt-4.1",

    instructions="You are a helpful agent that can search openai.com resources.",

    tools=[

        internet_search_agent.as_tool(

            tool_name="internet_search_agent", 

            tool_description="Use for searching web"

        )

    ],

)



result = await Runner.run(agent, "Search for OpenAI news")



# The parent only sees:

# - tool_call_item (calling internet_search_agent)

# - tool_call_output_item (text response)

# 

# But not the web_search_call from inside internet_search_agent

```



Is there a way to access the nested agent's tool calls?
