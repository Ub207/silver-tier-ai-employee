Here is a minimal code to replace the Tracing Processor: 

```

from agents.tracing import TracingProcessor



class CustomProcessor(TracingProcessor):

    def __init__(self):

        self.active_traces = {}

        self.active_spans = {}



    def on_trace_start(self, trace):

        print(f"Trace started: {trace}")

        self.active_traces[trace.trace_id] = trace



    def on_trace_end(self, trace):

        print(f"Trace ended: {trace}")

        del self.active_traces[trace.trace_id]



    def on_span_start(self, span):

        print(f"Span started: {span}")

        self.active_spans[span.span_id] = span



    def on_span_end(self, span):

        print(f"Span ended: {span}")

        # Process completed span

        del self.active_spans[span.span_id]



    def shutdown(self):

        print("Shutting down processor...")

        # Clean up resources

        self.active_traces.clear()

        self.active_spans.clear()



    def force_flush(self):

        print("Force flushing processor...")

    

from agents import Agent, Runner, set_trace_processors

 

set_trace_processors([CustomProcessor()])



agent = Agent(

  name="Assistant",

  instructions="Just say hello world",

)



result = Runner.run_sync(agent, "Hi")

print(result.final_output)



```



The Processor interface documentation is here:

https://openai.github.io/openai-agents-python/ref/tracing/processor_interface/



For example, here is Braintrust’s implementation:

https://github.com/braintrustdata/braintrust-sdk/blob/main/py/src/braintrust/wrappers/openai.py



And here is Langsmith’s implementation:

https://github.com/langchain-ai/langsmith-sdk/blob/main/python/langsmith/wrappers/_openai_agents.py



Most third-party monitoring platforms have integrations for the OpenAI Agents SDK, so you don’t need to implement it yourself.
