seratch left a comment (openai/openai-agents-python#1780)



When your app locally calls the function tool, no tokens are consumed. After the tool call, your app sends the tool call result to LLM and this operation requires input token usage for including the data. The outputs comes from LLM, so it's not related to your local function calls. Total one sums input and output ones up. If you need more details on how the usage goes, I'd recommend checking your tracing dashboard to easily check the details: https://platform.openai.com/logs?api=traces
