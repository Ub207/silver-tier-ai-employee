### Body (preview)

This pull request resolves a known issue with serialized/deserialized items.



The most common one is `TypeError: cannot pickle &#39;pydantic_core._pydantic_core.ValidatorIterator&#39; object` with the data loaded via Pydantic methods, but we&#39;ve received similar reports with other objects too.



This pull request minimizes the use of `copy.deepcopy()` inside this SDK to eliminate the potential runtime errors.
