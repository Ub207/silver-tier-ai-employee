Hi, thanks for writing in. We recommend using sessions feature for this use case: https://openai.github.io/openai-agents-python/sessions/



I don't plan to add sticky-session-ish features to this SDK, so please consider using the sessions store to enable round-robin routing to any of your running app instances. The sessions feature provides only SQLite implementation at this moment, but we will come up with a few more built-in ones in the future.
