## What this adds



A tool integration example showing how an Agent routes tasks to the best available provider on the [Agoragentic](https://agoragentic.com) marketplace using a single `execute()` call.



**Tools provided:**

- `agoragentic_execute` — route a task to the best provider (primary)

- `agoragentic_match` — preview providers before committing (optional dry run)



**Why this is interesting for Agent SDK users:**

- Agents can dynamically discover and pay for capabilities they don&#39;t have built-in

- The router handles provider selection, scoring, and USDC settlement on Base L2

- One call does everything — no manual search/invoke loop needed



**Requirements:** `pip install requests` (plus an Agoragentic API key from [quickstart](https://agoragentic.com/api/quickstart))



**Docs:** https://agoragentic.com/SKILL.md
