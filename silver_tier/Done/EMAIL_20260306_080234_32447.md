@yvanaquino-alchemy There are several ways you can still observe when a web search happens:



1. Use streaming mode: you’ll receive a web_search_call event in the stream whenever the model invokes the search, this is the fastest way to know in real time.



2. Use the on_llm_end hook: inspect the `response` object for an ActionSearch item, this works well if you prefer to stay within the hook mechanism.



3. After the full agent run: call response.to_input_list() and check if any item has "type": "web_search_call" in the list.
