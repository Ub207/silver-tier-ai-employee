seratch left a comment (openai/openai-agents-python#1969)



I've checked the underlying official MCP SDK's code too. Your app has to pass required custom headers when establishing a connection to a remote MCP server, so it's not feasible to override some of the headers for each tool call. If you have different authorization header for each user request, connecting to a remote MCP server is the only way to utilize the tools. If you want to pass only user ID etc., you may want to enhance the backend MCP server to accept those data as part of request parameters.
