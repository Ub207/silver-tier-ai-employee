Hi everyone - I'm at Microsoft - I was able to get this running. The main issue you're experiencing is that you're using a Beta Protocol URL with an SDK designed for the GA Protocol. We've been working to align the GA Protocol - so I recommend just using that. It can be used with preview or GA models. (Notice the path is openai/v1/realtime?model=xyz) Here's what I used: 



model_config : RealtimeModelConfig = {

            "api_key": os.getenv("AZURE_OPENAI_API_KEY", "xyz"),

            "url": f"wss://{os.getenv('AZURE_RESOURCE_NAME', 'myresourcename')}.openai.azure.com/openai/v1/realtime?model={os.getenv('AZURE_DEPLOYMENT_NAME', 'gpt-realtime828')}"

        }



        session_context = await runner.run(model_config=model_config)



Let us know if you run into any other difficulties.
