Thanks for writing in, but it seems your code snippet does not work. When you want to run an agent with tools + structured result, you can do something like this:



````python

import asyncio

from typing_extensions import TypedDict

from agents import Agent, Runner, function_tool



class EchoResult(TypedDict):

    reply: str



@function_tool

def echo(message: str) -> EchoResult:

    return EchoResult(reply="You said: " + message)



agent = Agent(

    name="TestAgent",

    instructions="You are a helpful assistant that can echo messages using the tools.",

    tools=[echo],

)



async def main():

    response = await Runner.run(agent, "Hello World")

    # print(response.raw_responses)

    print(response.final_output)



if __name__ == "__main__":

    asyncio.run(main())

```
