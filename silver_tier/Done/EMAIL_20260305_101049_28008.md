seratch left a comment (openai/openai-agents-python#1525)



Thanks for asking the question. 



It's not possible to hand off from a realtime agent to a text agent. A realtime agent handles WebRTC/WS connections plus it relies on realtime API's model, so text agents are not technically compatible. When you want to build a more complex agentic workflow as part of the session, you may want to build a text agent, which may hand off and/or call other sub agents, on the server side and use such text agents as tools of your realtime agent.



Hope this clarifies things.
