## Pull Request Overview



This PR fixes a resource leak in the VoiceStreamResult class by properly awaiting cancelled tasks during cleanup and preventing CancelledError from masking real exceptions.



**Key Changes:**

- Made `_cleanup_tasks()` async to properly await cancelled tasks using `asyncio.gather()` with `return_exceptions=True`

- Enhanced `_check_errors()` to check `task.cancelled()` before calling `task.exception()` to avoid CancelledError

- Updated the `stream()` method to await the now-async `_cleanup_tasks()`
