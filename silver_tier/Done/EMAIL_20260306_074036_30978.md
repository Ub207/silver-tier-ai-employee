### Question



I'm using the OpenAI Agents SDK and need clarification on token usage tracking. When I access token usage statistics through `result_obj.context_wrapper.usage`, I get:



```python

    "total_tokens": result_obj.context_wrapper.usage.total_tokens,

}
```

Are the tokens from step 2 (tool function calls) included in the `input_tokens`, `output_tokens`, and `total_tokens` counts?
