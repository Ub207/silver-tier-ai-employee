New (Beta): Temporal Support in Pydantic AI

-------------------------------------------



Pydantic AI can now run fully durable agents on Temporal - bringing reliability and long-lived coordination to your AI workflows:



* Survive API failures, exceptions, app restarts, and deploys

* Handle long-running, async, and human-in-the-loop interactions

* Preserve progress/state with replay-based fault tolerance



Why this matters: Temporal’s deterministic workflow model lets you pause coordination logic for hours or days and resume exactly where it left off - no lost state and no manual checkpointing. Non-deterministic work (LLM calls, tool calls, MCP server communication) is offloaded to activities with automatic retries.



Available now in v0.7.0 (beta). Temporal support will move to GA when we ship Pydantic AI v1 next month.
