You can now use Cursor Agent from the CLI or headless in any environment.



curl https://cursor.com/install -fsS | bash



If you're using Neovim, JetBrains, or other IDEs, you can now use the full power of Cursor Agent alongside your favorite tools.



The CLI works with any model as part of your Cursor subscription. You can now choose to use Cursor agent in the editor, or have multiple agents run in parallel in the terminal or remotely.



Cursor CLI interface



***********************

What Cursor CLI can do:

***********************



* Choose from any top AI model ( https://docs.cursor.com/en/models ) or use Auto ( https://docs.cursor.com/en/models#auto ) for speed and efficiency

* Work with the same rules and context as the Cursor IDE

* Connect external tools via

MCP ( https://docs.cursor.com/en/cli/mcp ) (Model Context Protocol)

* Run headless ( https://docs.cursor.com/en/cli/headless ) in scripts to automate code reviews, generate docs, and more

* Run it anywhere: GitHub Actions ( https://docs.cursor.com/en/cli/github-actions ), scripts, other IDEs, or any CI system

Try the CLI ( https://cursor.com/cli ) and let us know ( https://forum.cursor.com/ ) what features we should add.
