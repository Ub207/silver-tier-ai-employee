I don't think this is a bug. If a tool call exists in the convo and the agent thinks no need to call the tool again, it does not call. If you want to always call a tool, you can tell such in instructions or pass tool_choice=required;



```python

agent = Agent(

    name="Assistant",

    instructions="Reply very concisely. you must call get_current_time function to get the current time.",

    model_settings=ModelSettings(

        tool_choice="required",

    ),

    tools=[get_current_time],

)

```
