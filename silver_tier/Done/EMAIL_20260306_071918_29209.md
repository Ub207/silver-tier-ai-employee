@seratch I've managed to get it working.



I think there is a bug in the SDK where it's not using the provided API Key from `model_config` .



```python

        runner = RealtimeRunner(agent)

        session_context = await runner.run(

            model_config={

                "api_key": settings.azure_openai_api_key,

                "url": f"wss://****.openai.azure.com/openai/realtime?api-version=2025-04-01-preview&deployment=gpt-4o-mini-realtime-preview&api-key={settings.azure_openai_api_key}",

            }

        )

```



When passing in the API Key as URL Query parameter it seems to work!
