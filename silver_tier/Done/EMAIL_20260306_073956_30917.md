To update you, I tried using lite-llm based in `google-adk` with similar code, turns out it's able to handle the scenario pretty well.



```python

import datetime

from zoneinfo import ZoneInfo

from google.adk.agents import Agent

from google.adk.models.lite_llm import LiteLlm



from pydantic import BaseModel, Field



import litellm



litellm.use_litellm_proxy = True



class WeatherOutput(BaseModel):

    is_good_weather: bool = Field(..., description='Is the weather good for travel or outdoor activitied?')

    details: str = Field(..., description='Give the detailed weather report.')



def get_weather(city: str) -> str:

    """Retrieves the current weather report for a specified city.



    Args:

        city (str): The name of the city for which to retrieve the weather report.



    Returns:

        str: Weather status

    """

    return f'Current weather in {city} is jublious like a rainbow.'



model = LiteLlm(model="moonshotai/kimi-k2-instruct")

root_agent = Agent(

    name="Weather_Agent",

    # model="gemini-2.0-flash",

    model=model,

    description=(

        "Agent to answer questions about weather in a city."

    ),

    instruction=(

        "You are a helpful weather assistant. "

        "Always call the available tools to gather weather data. "

        "Do not fabricate answers. "

        "Only use the structured output (WeatherOutput) after tool results are retrieved."

    ),

    tools=[get_weather],

    output_schema=WeatherOutput

)

```
