### Describe the feature



Hello! As the docs mention, durable execution can be used for long running agents and human-in-the-loop.



I recently worked on a [Pydantic AI + DBOS integration](https://ai.pydantic.dev/durable_execution/dbos/), where the agent's main loop automatically runs as a durable workflow and external model calls are wrapped as steps. A similar approach could work here. DBOS is just a library (supports SQLite/Postgres), so you can add durable execution without extra infrastructure and still run everything inside the application process.



Previously, I also built an example combining Swarm and DBOS, [Durable Swarm](https://github.com/dbos-inc/durable-swarm). Since this SDK is a production-ready upgrade of Swarm, I expect many of the same patterns will apply.



Happy to discuss an integration plan. I'd be glad to help lead the implementation.
