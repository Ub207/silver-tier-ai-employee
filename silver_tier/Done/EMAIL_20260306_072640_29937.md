Pydantic AI v1

--------------



What v1 means: After nine months of feedback, iteration, and real-world usage (and a stunning 15 million downloads!), we're confident in the framework's API and ready to commit to its stability, so that you can keep building great GenAI applications without having to worry about the foundation shifting under you.



New in v1:



* Human-in-the-Loop tool approval ( https://ai.pydantic.dev/deferred-tools/ ) for safe, supervised actions

* Durable Execution with Temporal ( https://ai.pydantic.dev/durable_execution/temporal/ ) production-ready (out of beta)



What's next in v1.x: prompt caching, handoffs, durable execution for graphs, embeddings support, MCP resource support, and context-free Grammar output support.



View Documentation → ( https://ai.pydantic.dev/ )
