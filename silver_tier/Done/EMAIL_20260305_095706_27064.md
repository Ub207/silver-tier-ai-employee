Unfortunately, handoff design is supposed to be used only for agent A to agent B delegation at a time. So, I would suggest going with a mix of agents and deterministic dispatch logic like some of the examples (e.g., financial research agent etc.). This means you can define a planner agent and ask the agent to produce plans to run. Then your subsequent code can run all the plans either in parallel or sequentially. Hope this helps.
