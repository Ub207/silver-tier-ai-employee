### Body (preview)

This pull request introduces a new optimization and evaluation module inspired by DSPy, providing tools for evaluating and improving LLM agents. It includes comprehensive documentation and real-world examples for both text and structured outputs, such as few-shot selection and instruction optimization. The changes also add examples for email triage and structured calendar event extraction, demonstrating the new utilities in practice.



**Documentation and User Guidance:**



* Added detailed documentation in `docs/optimizers.md` and `docs/ja/optimizers.md` describing the new optimizer utilities (e.g., `evaluate_agent`, `cross_validate_agent`, `BootstrapFewShot`, `InstructionOptimizer`), their API, and usage for both text and structured outputs. Includes code examples and metric customization.
