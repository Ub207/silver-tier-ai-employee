Thanks for writing in. When you pass the customization option this way, you can already use mini model for your realtime agents:

```python

from agents.realtime import RealtimeRunner

from agents.realtime.config import RealtimeRunConfig, RealtimeSessionModelSettings



agent = build_my_agent()

config = RealtimeRunConfig(

    model_settings=RealtimeSessionModelSettings(

        model_name="gpt-realtime-mini",

    ),

)

runner = RealtimeRunner(agent, config=config)

```



You can modify this example to quickly see how it works: https://github.com/openai/openai-agents-python/tree/main/examples/realtime/app
