### Body (preview)

Fixed an import error in the **OpenAI Agents SDK** where the code was attempting to import a non-existent type:



```python

from openai.types.responses import ResponseTextConfigParam

```



In recent versions of the `openai` Python library, this type no longer exists and has been replaced by:



```python

from openai.types.responses import ResponseFormatTextConfigParam

```



#### Error Before



Running any agent with the current SDK results in:



```

ImportError: cannot import name &#39;ResponseTextConfigParam&#39; from &#39;openai.types.responses&#39;

```



#### Fix Applied



* Replaced `ResponseTextConfigParam` with the correct `ResponseFormatTextConfigParam` import.

* Verified functionality by running a simple agent with `OpenAIChatCompletionsModel`, error is resolved and responses are returned successfully.
