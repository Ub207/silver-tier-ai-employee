Probably, you're using OpenAI's Responses API for other agents and doing handoffs or whatever to pass the conversation to the Claude model. Unfortunately, mixing Responses API and Chat Completions API models do not work. Consider using only Chat Completions for all agents or using only OpenAI's Responses API. See also: https://github.com/openai/openai-agents-python/issues/1485#issuecomment-3190879185
