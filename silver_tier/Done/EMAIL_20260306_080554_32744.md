@muneebarshadd Because you attached the hooks only to `my_agent`, when the handoff happens, control is transferred to `next_agent`, which doesn’t have any hooks attached, so your on_end and on_llm_end methods will no longer run.



Try this instead 👇



```

next_agent = Agent( name="next_agent", instructions="...",  hooks=MyAgentHooks(),  # 👈 add this

)

```



That way, both agents have their own AgentHooks, and you’ll see the hooks being printed.



To clarify a bit more:



1. AgentHooks only apply to the specific agent they’re attached to. Once a handoff occurs, the new agent needs its own hooks if you want to observe its lifecycle.

2. RunHooks apply to the entire run, even if handoffs happen internally.

3. If your goal is to observe the full run, it’s usually better to use RunHooks.
