Securely connect AI tools to your app

Clerk now supports the Model Context Protocol (MCP) in Express.js applications. This lets your users authorize AI applications

like Claude and Cursor to access their data in your app through secure OAuth flows.

Implementation requires as little as 50 lines of code in a single file. The setup uses Clerk's OAuth provider with built-in MCP

handlers for authentication, protected resources, and server metadata. Once running, AI tools connect with just a URL

configuration. No stdio tools, command execution, or additional dependencies needed.

 * Minimal code: Complete server in ~50 lines using Clerk's MCP tools (shown below)

 * Standard OAuth: Uses RFC-compliant OAuth flows with automatic token management

 * Framework-agnostic: Works with existing Express apps and middleware
